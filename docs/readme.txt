本文件夹（docs）说明
====================

【文件夹作用】
本目录存放 AI FranchTeacher 项目的说明与运维文档，涵盖部署、认证、云端发布、目录结构说明及故障排查，便于开发、部署与日常维护时查阅。

【各文件说明】

readme.txt
  本说明文件：概述 docs 目录用途，并逐条介绍本目录下各文档文件。

README.md
  文档中心总索引：汇总文档导航、按场景选读指引、系统端口与架构概览等，适合作为进入其他文档的入口。

PROJECT_STRUCTURE.md
  项目目录结构说明：说明仓库（含前端、后端、数据库脚本等）的组织方式与各路径职责，便于理解代码布局。

DEPLOYMENT_GUIDE.md
  完整部署指南：从系统要求、安装准备、数据库与后端、前端部署到启动测试、性能优化与生产环境配置的详细步骤。

AUTH_SETUP_GUIDE.md
  用户认证系统部署与使用指南：数据库设置、后端与前端相关配置、pgAdmin 管理、REST API 说明及常见问题。

RENDER_DEPLOYMENT.md
  Render 云端部署指南：将认证服务、数据库与 GitHub Pages 前端联调的架构说明与部署步骤。

TROUBLESHOOTING.md
  故障排查手册：服务状态检查思路、常见问题分类与解决步骤、维护相关命令等。
